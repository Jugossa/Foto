# Evento de Fotos con Moderación

## Cómo ejecutar en tu PC

1. Instalar dependencias:
   npm install

2. Ejecutar el servidor:
   node index.js

3. Abrir en tu navegador:
   - http://localhost:3000/upload → para subir fotos
   - http://localhost:3000/moderar → para aprobar/rechazar
   - http://localhost:3000/galeria → para mostrar en loop en pantalla gigante

⚠️ Asegúrate de tener el archivo credentials.json en la raíz del proyecto.
